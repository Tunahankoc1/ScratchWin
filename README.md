# Scratch & Win 🎰

A scratch card mini app on Base. Scratch to reveal your prize — wins are recorded on-chain.

🌐 **Live:** [scratch-win-iota.vercel.app](https://scratch-win-iota.vercel.app)

---

## How It Works

1. Connect your Coinbase Wallet
2. Scratch the card to reveal your prize
3. Win → transaction recorded on Base

## Prizes

| Prize | Amount | Chance |
|-------|--------|--------|
| 💎 Jackpot | 100 USDC | 3% |
| 🥇 Gold | 10 USDC | 8% |
| 🎁 Mystery | 5 USDC | 14% |
| ⭐ Lucky | 1 USDC | 20% |

## Stack

- HTML + CSS + JavaScript
- Coinbase Wallet (Base Mainnet)
- Farcaster Frame SDK
- Vercel

## Setup

```bash
git clone https://github.com/Tunahankoc1/ScratchWin
cd ScratchWin
npm install
npm run dev
```

Open [https://scratch-win-iota.vercel.app](https://scratch-win-iota.vercel.app)
